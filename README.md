# 剑域幸存者 · 手机豪华版

一款可直接在浏览器运行的国风修仙 Roguelite 生存小游戏。玩家操控剑修在持续涌来的兽潮中移动、自动御剑攻击、吸收修为并选择强化，每隔 45 秒迎战一位 Boss，不断突破更高天劫。

当前版本：`V2.7.4_fix6`

## 在线游玩

- 正式地址：<https://wch135938-creator.github.io/sword-survivor-mobile/>
- 当前版本直达：<https://wch135938-creator.github.io/sword-survivor-mobile/?v=2.7.4_fix6>

建议手机横屏游玩。首次更新版本后，如仍显示旧内容，可刷新页面或重新打开版本直达链接。

## 核心玩法

- 角色自动攻击最近敌人，玩家主要负责移动和走位。
- 击杀敌人获得修为，升级后从三个强化选项中选择一个。
- 每一劫持续 45 秒，倒计时结束后 Boss 登场。
- 击败 Boss 后进入下一劫，并自动保存当前成长进度。
- Boss 会随关卡推进提高生命、伤害和弹幕复杂度。

## 操作方式

### 电脑

- `WASD` 或方向键：移动
- `空格`：暂停 / 继续
- `R`：重新开始

### 手机

- 左侧动态摇杆或在战斗区域按住拖动：移动
- 右侧按钮：暂停、重开、声音和说明
- 推荐横屏使用

## Boss 弹幕成长

| 关卡范围 | 弹幕表现 |
| --- | --- |
| 第 1～3 劫 | 单发子弹 |
| 第 4～5 劫 | 双发子弹 |
| 第 6～10 劫 | 三发扇形 |
| 第 11～20 劫 | 强化但保持低频 |
| 第 21 劫以后 | 环形、追踪与多阶段组合弹幕 |

高阶 Boss 会根据生命比例进入强化阶段和终局暴走阶段。

## 强化与奖励

游戏中可获得或强化：

- 飞剑伤害、攻击速度、多重飞剑和穿透
- 移动速度、最大气血、暴击和护甲
- 护体灵剑、紫霄天雷、回血和修为吸取
- 狂暴符、防护罩、诛仙令和僚机令

### 特殊奖励

- **回血丹**：恢复气血
- **狂暴符**：短时间提高火力
- **防护罩**：抵挡一次攻击后消失
- **诛仙令**：发动全屏诛仙剑阵；每个 Boss 阶段随机出现 1～2 个
- **僚机令**：召唤或升级僚机

## 僚机系统

- 最多同时拥有 3 架僚机。
- 僚机最高 30 级。
- 升级会提高伤害、攻击速度，并逐步解锁散射、重炮和导弹爆裂形态。
- 僚机达到 30 级后，再拾取“机”不再升级，每个转化为 `+5` 分。

## 计分规则

| 目标 | 分数 |
| --- | ---: |
| 普通小怪 | 1 分 |
| 精英怪 | 2 分 |
| Boss | 10 分 |
| 满级后额外僚机令 | 5 分 |

最高分保存在当前浏览器本地。

## 自动存档

`V2.7.4_fix6` 已加入关卡存档功能：

- 每击败一个 Boss 自动保存。
- 拾取奖励或选择强化后，会同步更新当前存档内容。
- 下次打开游戏时可选择“继续历练”或“重新开始”。
- 继续游戏会从已完成关卡的下一劫开始，并重新进行 45 秒倒计时。
- 会保留角色等级、修为、强化、攻击属性、僚机、分数和击杀数。
- 继续时气血恢复至满值，当前关卡中的敌人、子弹和掉落不会保存。

存档使用浏览器 `localStorage`，只保存在当前设备和当前浏览器中。清理浏览器数据、更换浏览器或更换设备后，存档不会自动迁移。

## PWA 与离线运行

项目包含：

- `index.html`：完整游戏
- `manifest.json`：PWA 配置
- `sw.js`：离线缓存和版本更新
- `VERSION.txt`：当前版本号

在线打开一次后，支持在网络不稳定或离线环境下继续加载已缓存版本。不同浏览器对 PWA 安装入口的显示方式可能不同。

## 本地运行

下载仓库后，可直接打开 `index.html` 进行基础游玩。

为了完整测试 Service Worker、PWA 和缓存功能，建议通过本地 HTTP 服务运行，例如：

```bash
python -m http.server 8080
```

然后访问：

```text
http://localhost:8080/
```

## GitHub Pages 部署

仓库使用 GitHub Pages 发布。更新版本时应一起覆盖：

```text
index.html
manifest.json
sw.js
VERSION.txt
```

同时统一修改页面版本号、Manifest 版本参数和 Service Worker 缓存名称，避免浏览器继续加载旧版本。

## 数据与隐私

- 不上传游戏数据。
- 不依赖远程游戏服务器。
- 最高分和关卡存档仅保存在浏览器本地。
